# 🚀 快速启动指南

## 第一次使用？

欢迎！这是一个完整的个人介绍网站项目。按照以下步骤快速启动：

## 📋 前置要求

确保你的电脑已安装：
- Node.js (版本 14 或更高)
- npm (通常随 Node.js 一起安装)

检查版本：
```bash
node --version
npm --version
```

## 🎬 启动步骤

### 1. 进入项目目录
```bash
cd personal-website
```

### 2. 安装依赖（首次运行）
```bash
npm install
```

这会安装所有需要的包，可能需要几分钟。

### 3. 启动开发服务器
```bash
npm start
```

### 4. 查看网站
浏览器会自动打开 http://localhost:3000

如果没有自动打开，手动在浏览器中访问这个地址。

## 🎮 使用指南

### 网站导航
1. **加载动画**: 点击任意位置或按空格键跳过
2. **晨间仪式**: 依次点击三个步骤，然后选择路径
3. **认知地图**: 悬停查看节点，点击展开详情
4. **感官漫游**: 滚动查看音符流动和咖啡动画
5. **声音档案**: 悬停唱片，点击分区查看歌词
6. **构建日志**: 滚动查看时间轴条目
7. **联结邀请**: 点击复制微信号

### 主题切换
点击右上角的太阳/月亮图标切换晨间/深夜模式。

### 进度条
左侧进度条显示当前时间和完成进度。

## 🛠️ 其他命令

### 运行测试
```bash
npm test
```

### 构建生产版本
```bash
npm run build
```

构建完成后，文件会在 `build/` 目录中。

### 代码检查
```bash
npm run lint
```

## 🎨 自定义

### 修改个人信息
编辑 `src/components/MorningRitual.tsx` 中的个人信息卡片内容。

### 修改音乐分区
编辑 `src/components/SoundArchive.tsx` 中的 `musicSections` 数组。

### 修改日志条目
编辑 `src/components/BuildLog.tsx` 中的 `logEntries` 数组。

### 修改微信号
编辑 `src/components/Connection.tsx` 中的微信号。

### 修改主题颜色
编辑 `src/styles/variables.css` 中的 CSS 变量。

## 🐛 常见问题

### 端口被占用
如果 3000 端口被占用，可以：
1. 关闭占用端口的程序
2. 或者使用其他端口：
```bash
PORT=3001 npm start
```

### 依赖安装失败
尝试清除缓存后重新安装：
```bash
npm cache clean --force
npm install
```

### 页面空白
1. 检查浏览器控制台是否有错误
2. 确保所有依赖都已安装
3. 尝试重启开发服务器

### 动画卡顿
1. 关闭浏览器的其他标签页
2. 检查电脑性能
3. 尝试切换到性能更好的浏览器（推荐 Chrome）

## 💡 提示

### 开发技巧
1. 修改代码后会自动刷新页面
2. 使用浏览器开发者工具查看元素和调试
3. 按 F12 打开开发者工具

### 性能优化
1. 生产构建会自动优化代码
2. 图片使用合适的格式和大小
3. 避免在动画中使用复杂的计算

### 部署建议
1. 使用 `npm run build` 构建生产版本
2. 可以部署到 Vercel、Netlify、GitHub Pages 等平台
3. 确保配置正确的环境变量
